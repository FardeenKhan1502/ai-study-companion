# AI Study Companion 🚀

An AI-powered platform that helps students study smarter by converting syllabus PDFs into structured learning plans, recommending relevant videos, and providing an intelligent chatbot for doubt solving.

## 🔥 Features

* 📄 Upload syllabus PDF and extract topics
* 🎥 Smart video recommendations
* 🤖 AI chatbot (Gemini API)
* 📊 Study roadmap
* ⚡ Exam Mode

## 🧠 Tech Stack

* React.js (Frontend)
* Node.js + Express (Backend)
* Gemini API (AI)
* pdf-parse

## 🚀 How to Run

### Backend

cd backend
npm install
npm run dev

### Frontend

cd frontend
npm install
npm run dev
